# CSE3120-Contest
contest repository for CSE3120 game. Group members Eddie Di Bella, Jacob Ewasko

## Compile/Run instructions
- build the project in visual studio
- open a terminal window
- navigate to the debug folder inside of the project folder
- run the .exe file project.exe

## GAMEPLAY INSTRUCTION
- use W A S D to navigate the snake around the map
- if the game is over press r to start another game
